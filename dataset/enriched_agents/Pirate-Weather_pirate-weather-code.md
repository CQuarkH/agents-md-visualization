# AGENTS Instructions

This repository uses `ruff` for linting and formatting. Run lint checks with:

```bash
scripts/lint
```

Format the code with:

```bash
scripts/format
```

Run the test suite using `pytest` before submitting changes:

```bash
pytest
```
